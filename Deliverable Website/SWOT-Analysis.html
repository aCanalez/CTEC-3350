SWOT Analysis

Internal Strengths Weaknesses
  • Knowledge
    • Company has been building fences since 1976

      How to capitalize: We can offer free professional assessment of current fence condition, which can lead to selling services.
    • Company is great at fence projects

      How to capitalize: We can make this a main point through all services page.
    • Company is already actively using social media

      How to capitalize: We can offer a small discount such as 3% if you like and share our page. Which will lead to great outreach and more customers.
  • Versatility
    • We build fences from all types of materials
    • We also offer sidewalk, outdoor kitchens, and tennis court resurfacing
  • BBB Accredited
    • Develops visitor trust on brand
    • Adds credibility to website by showing BBB badge
  • Lack of effective call to action
    • Website could use a call to action button.
    • Website lacks an engaging theme and visual appeal

      Recommendation: Website redesign to incorporate large call to actions for service.
  • Lack of user conversions
    • Site uses an image to display phone number, this should be a clickable element
    • Mobile phone users want to click a number and call

      Recommendation: Website redesign will make the number actual text and clickable to website visitors
  • Navigation
    • User can become lost on mobile due to weird mobile menu
    • Some elements can be overlooked on mobile
    • Website does not offer suggestions on what to browse next

      Recommendation: Website will incorporate suggestions to related products when user visits a product page
External Opportunities Threats
  • Improve user experience
    • Use new technologies HTML5, CSS3, CDN
    • Use new web design trends

      Advantage: Deliver a flawless user experience on mobile which makes up to 56% of website traffic in current market
  • Effective Marketing
    • Effective call to actions
    • Update content making it more relevant in process

      Advantage: By improving call to actions user conversion can happen sooner
  • Social
    • Uplift image on social media.
    • Post to social media more frequently.

      Advantage: An online presence increases a websites credibility online and we can use this to drive visitors to our website.
  • Spam (highly probable)
    • Include an easy captcha method
    • Suppression: Block IP address from spam and include a captcha
  • Competitors (not likely)
    • Competitors copying features
    • Competitor already using our planned solutions
    • Advertisement budget
  • Customer Needs
    • Customers looking for particular product we don't offer
    • Customer not able to access our website due to incompatibility

      Suppression: Allow users to inquire about services and recommending to us
