Typography

Headings

Normal Headings

All HTML headings, <h1> through <h4>, are available.

Heading Example
<h1></h1>

h1. Heading 1
<h2></h2>

h2. Heading 2
<h3></h3>

h3. Heading 3
<h4></h4>

h4. Heading 4

Customized Headings

These are pseudo-headings that use the paragraph tag with a class of <h1>, <h2>, <h3>, or <h4>. These headings are used mainly for look but they DO NOT denote importance as the normal <h1> through <h4> headings for SEO purposes.

Heading Example
<p class="h1"></p>

h1. Heading 1
<p class="h2"></p>

h2. Heading 2
<p class="h3"></p>

h3. Heading 3
<p class="h4"></p>

h4. Heading 4

Inline Text Elements

Element Markup Example
<mark></mark> The HTML5 specification defines the "mark" element as an element which represents a run of text in one document marked or highlighted for reference purposes. Mark element is used to tag content which has relevance.
<del></del> The <s> element is redefined in HTML5 to use the <del> element instead. This is now used to define text that is no longer correct. Use the <s>.
<u></u> The <u> tag represents some text that should be stylistically different from normal text, such as misspelled words or proper nouns in Chinese.
<b></b> & <strong></strong> The <b> tag specifies bold text.
The <strong> tag is a phrase tag. It defines important text.
<i></i> & <em></em> The <i> tag defines a part of text in an alternate voice or mood. The content of the tag is usually displayed in italic.
The <em> tag is a phrase tag. It renders as emphasized text.
<q></q> The <q> tag defines a short quotation. Browsers normally insert quotation marks around the quotation.

Quotes

For quoting blocks of content from another source within your document. Wrap <blockquote> around any HTML as the quote.
